## 🚀 Başlangıç için Gerekli Bilgiler

KUMBO projesini başlatmak için lütfen aşağıdaki bilgileri sağlayın:

### 1. İş Fikri Detayları
- [ ] **Ana Fikir:** KUMBO ne yapacak? (Detaylı açıklama)
- [ ] **Hedef Kitle:** Kimler için yapılıyor?
- [ ] **Ana Problem:** Hangi sorunu çözüyor?
- [ ] **Çözüm:** Nasıl çözüyor?

### 2. Pazar Bilgileri
- [ ] **Hedef Pazar:** Hangi pazar/coğrafya?
- [ ] **Pazar Büyüklüğü:** TAM/SAM/SOM (varsa)
- [ ] **Rakipler:** Bilinen rakipler kimler?

### 3. Teknik Tercihler
- [ ] **Platform:** Web / Mobile / Her ikisi?
- [ ] **Tech Stack:** Tercihiniz var mı? (React, Next.js, vs.)
- [ ] **Özellikler:** Olmazsa olmaz özellikler neler?

### 4. İş Modeli
- [ ] **Gelir Modeli:** Nasıl para kazanacak?
- [ ] **Fiyatlandırma:** Freemium / Subscription / One-time?
- [ ] **Lansman Hedefi:** Ne zaman canlıya çıkmak istiyorsunuz?
